## Statements

Types of statements:
* if 
* else
* elif 

How to write these statements?
```python
x = 5
if x < 5:
 print("X is less than 5")
elif x == 5:
   print("X is equal to 5")
else:
    print("X is equal to " + x)
```
This code checks if x is less than 5, equal to 5, and more than 5.
